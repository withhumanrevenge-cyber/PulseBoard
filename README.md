## Getting Started

1. **Environment Config**
   Create a `.env.local` file with:
   - `NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY`
   - `CLERK_SECRET_KEY`
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`

2. **Dependencies**
   ```bash
   npm install
   ```

3. **Development**
   ```bash
   npm run dev
   ```

## Key Features
- **Zero-Ops Updates:** Automated sync between GitHub activity and public profiles.
- **Glassmorphism UI:** Premium, animated interface optimized for high-end portfolios.
- **SSR Optimized:** Server-side fetching for instant data delivery and SEO.

## Deployment
Optimized for Vercel. Ensure environment variables are mirrored in the project settings.
